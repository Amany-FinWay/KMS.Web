Download Kiosk Agent

The Kiosk Agent is required for registering and managing kiosks. Download and install it on each kiosk terminal. After installation, the kiosk will automatically register with the portal and report its hardware details and capabilities.

Installation Steps

  1. 1. Download the appropriate agent version for your kiosk platform
  2. 2. Run the installer on the kiosk terminal with administrator privileges
  3. 3. Follow the installation wizard and configure the portal connection
  4. 4. The agent will automatically register the kiosk and begin reporting status
  5. 5. Verify the kiosk appears in your dashboard within 2-3 minutes

Download Digital Signage Agent

The Digital Signage Agent enables remote content management and display control on digital screens. Install this agent on devices connected to your digital signage displays to enable centralized content deployment, scheduling, and real-time monitoring.

Installation Steps for Digital Signage

  1. 1. Download the digital signage agent for your display platform
  2. 2. Install the agent on the device connected to your display screen
  3. 3. Configure the portal connection and screen identification
  4. 4. The agent will register the screen and begin listening for content
  5. 5. Deploy content, schedules, and templates from the Digital Signage module
